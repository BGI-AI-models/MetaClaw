# Effect Sizes and Power

Use this reference when reporting magnitude, precision, or planning sample size.

## Why Effect Sizes Matter

- p-values do not show magnitude.
- Large samples can make trivial effects significant.
- Effect sizes support interpretation and meta-analysis.
- Power analysis helps with study planning.

## Common Effect Sizes

### T-tests and mean differences
- Cohen's d
- Hedges' g for small samples
- Glass's delta when a control SD is the right reference

### ANOVA
- Eta-squared
- Partial eta-squared
- Omega-squared
- Cohen's f

### Correlation
- Pearson's r
- Spearman's rho

### Regression
- R² and adjusted R²
- Standardized coefficients beta
- f² for nested model comparison

### Categorical data
- Cramér's V
- Phi for 2x2 tables
- Odds ratio and risk ratio when appropriate

### Bayesian summaries
- Bayes factors
- Credible intervals
- Posterior probabilities

## Power Analysis

### A priori power analysis
Use this before data collection to estimate required sample size.

### Sensitivity analysis
Use this after data collection to describe what effect size the study could detect.

### Caution
Post hoc power is usually not helpful and should not replace uncertainty reporting.

## Reporting Rules

- Always report effect sizes with confidence intervals when possible.
- Pair effect sizes with the relevant test statistic and p-value.
- Distinguish statistical significance from practical importance.
- Report null results with the same care as significant ones.
