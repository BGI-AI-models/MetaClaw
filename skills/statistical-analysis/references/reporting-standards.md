# Statistical Reporting Standards

Use this reference for APA-style or publication-ready statistical reporting.

## General principles

1. Be transparent.
2. Be complete.
3. Report non-significant findings and violations.
4. Write for the reader.
5. Keep the analysis reproducible.

## Methods section

Report:
- design
- participants or data source
- measures or variables
- procedure
- software and versions
- significance threshold
- assumption checks
- missing data handling
- multiple comparison corrections
- effect size measures

## Results section

### Descriptives
Report sample size, central tendency, variability, and confidence intervals when useful.

### Assumption checks
State which assumptions were tested, how they were checked, and whether they were satisfied.

### Inferential statistics
Report:
- test statistic
- degrees of freedom when relevant
- exact p-value when possible
- effect size
- confidence interval
- direction of effect

## Example patterns

### T-test
Group A scored higher than Group B, t(df) = value, p = value, d = value, 95% CI [lower, upper].

### ANOVA
There was a main effect of condition, F(df1, df2) = value, p = value, eta squared = value.

### Correlation
There was a correlation between X and Y, r(df) = value, p = value, 95% CI [lower, upper].

### Regression
The model was significant, F(df1, df2) = value, p = value, R² = value. Predictors are reported with B, SE, beta, t, p, and confidence intervals.

### Bayesian results
Report priors, posterior estimates, credible intervals, Bayes factors, and convergence diagnostics.

## Null results

Say that there was no significant difference or no evidence for a relationship, rather than claiming absolute equivalence unless the design supports that claim.

## Common mistakes to avoid

- reporting p = .000 instead of p < .001
- omitting effect sizes
- ignoring assumption checks
- mixing statistical and practical significance
- overstating null results
